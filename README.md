# ML-Social-Bot-Detector
Python-инструмент для обнаружения ботов в социальной сети ВКонтакте с использованием методов машинного обучения и нейронных сетей. Проект предназначен для анализа профилей пользователей по метаданным и выявления автоматизированных аккаунтов.
## 🚀 Возможности
- 🤖 Сбор данных пользователей через официальный VK API
- 📊 Извлечение ключевых метрик: друзья, подписчики, посты на стене, статус аккаунта
- 🧠 Обучение и сравнение нескольких моделей машинного обучения:
  - Дерево решений
  - Случайный лес
  - Наивный Байес
  - Градиентный бустинг
- 🔗 Реализация нейронной сети "Многослойный Персептрон" на TensorFlow/Keras
- 📈 Автоматическая обработка пропущенных данных и удаление дубликатов
- 📉 Сравнительный анализ точности моделей в табличном виде
## 📦 Установка
1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/your-username/social-media-bot-detection.git
   cd social-media-bot-detection
   ```
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
   Основные зависимости: vk-api, pandas, numpy, scikit-learn, tensorflow, keras
3. Настройте доступ к VK API
   Получите токен доступа ВКонтакте
   Замените строку в функции generateVKAuthToken() на ваш токен

## ⚙️ Параметры и метрики
Используемые признаки:
- friends_cnt — количество друзей
- followers_cnt — количество подписчиков
- subscriptions_cnt — количество подписок
- wall_posts_cnt — число записей на стене
- is_deleted — флаг удаленного аккаунта
- is_banned — флаг заблокированного аккаунта
Целевая переменная:
- is_bot — 1 (бот) / 0 (реальный пользователь)
## 🔧 Технологии
- Python 3.8+ — основной язык программирования
- vk-api — взаимодействие с API ВКонтакте
- pandas / numpy — обработка и анализ данных
- scikit-learn — реализация ML-алгоритмов и предобработка
- TensorFlow / Keras — построение и обучение нейронной сети
- CSV / JSON — форматы хранения данных
